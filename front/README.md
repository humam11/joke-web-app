# FunnyHub Frontend

Инструкция по запуску фронтенд-части проекта.

## Требования

- Node.js 18 или новее
- npm

## Установка зависимостей

Перейдите в папку фронта:

```bash
cd front
```

Установите зависимости:

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:5173
```

Если порт `5173` занят, Vite автоматически предложит другой порт.

## Сборка проекта

```bash
npm run build
```

Готовая сборка появится в папке `dist`.

## Предпросмотр сборки

```bash
npm run preview
```
