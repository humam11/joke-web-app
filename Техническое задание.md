FunnyHub — сайт с анекдотами и развлечениями

FunnyHub — это веб-приложение, разработанное на основе React JS, предназначенное для публикации и просмотра развлекательного контента: анекдотов, шуток, историй и других материалов. Пользователи могут просматривать контент, переходить по категориям и получать быстрый доступ к новым публикациям.

Команда проекта ЭУ - 230 Участники: Зеркаль Дмитрий - Dommonster Хумам Ибрахим - humam11 Антоненко Данил - thaiti Арапов Михаил - meblessme

Цель проекта Создание удобного веб-приложения для просмотра и публикации развлекательного контента с возможностью масштабирования функционала.

Задачи проекта

Реализовать веб-сайт на React JS
Создать структуру базы данных для хранения контента
Реализовать отображение анекдотов и категорий
Обеспечить удобную навигацию по сайту
Целевая аудитория

Пользователи интернета 14+
Люди, желающие отвлечься и отдохнуть
Любители юмористического контента
Проблема

Недостаток простых и удобных сайтов с быстрым доступом к юмору
Перегруженность аналогичных платформ лишним функционалом
Отсутствие централизованного хранения короткого развлекательного контента

Описание работы приложения Тип проекта: веб-приложение

Приложение позволяет:

Просматривать список анекдотов
Открывать отдельные записи
Фильтровать контент по категориям
Управлять контентом через административную панель
Архитектура построена на React JS с использованием шаблонов и базы данных.
